# Heart Disease Prediction Project

## Overview
This project aims to develop a machine learning model that can predict the likelihood of heart disease based on various health metrics. The model is trained on a dataset containing medical attributes from patients.

## Installation Instructions
To set up this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/BahrouniAyoub/Predicting-Heart-Disease.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Predicting-Heart-Disease
   ```
3. Install the required dependencies using pip:
   ```bash
   pip install -r requirements.txt
   ```

## Usage Guide
To use the heart disease prediction model:
1. Ensure all dependencies are installed as listed in `requirements.txt`.
2. Run the main application script:
   ```bash
   python app.py
   ```
3. You can input patient metrics to receive a prediction on heart disease likelihood.

## Project Details
- **Dataset Used**: The model is trained using publicly available heart disease datasets, such as the Cleveland Heart Disease dataset.
- **Technologies**: This project uses Python with libraries such as Pandas, NumPy, Scikit-learn, and Flask for the web application.
- **Model**: Various machine learning algorithms are tested, including Logistic Regression, Decision Trees, and Random Forests.
- **Future Work**: The project could be expanded to improve accuracy, include more features, or deploy the model as an API.
